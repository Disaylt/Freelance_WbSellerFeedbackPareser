В корневой папке с запускаемые .exe файлом проекта находятся 2 папки - files, settings.

В папку files закидывается Excel файл с сообщениями в тех поодержку. (Важно при запуске программы и обновления excel файло, он должен быть закрыт)

В папке settings находится файл для подключения селлеров. Файл открывается любый текстовым редактором.
Структура файла
Если 1 селлер - [{data}]
Если 2 и больше - [{data},{data},{data}]
Где data - данные сееллера для парсинга и записи в excel.

Пример файла [{"SellerName":"Test","WbToken":"123123","SupplierId":"435345345"}]
или [{"SellerName":"Test1","WbToken":"123123","SupplierId":"435345345"}, {"SellerName":"Test2","WbToken":"123123","SupplierId":"435345345"}] - обратите внимание где находится запятая.

SellerName - произвольно название вашего селлера (Этим название подписывается лист в Excel)
WbToken и SupplierId - находятся в куках вашего браузера, зайдите на необходимого селлера, после чего скопируйте значения данных кук. 

Далее запустите exe файл, в логах будет писаться основные действия программы. В конце будет написано что Excel файл сохранен либо консоль закроется автоматически сохраанив файл.
